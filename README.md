# Entrega Tudo

## Dependencia

É preciso ter o gcc instalado e o c++17

## Como executar

1º passo: Compilar

```sh
make build
```

2º passo: executar

```sh
./bin/main instancia opcao [?seed]
```

opcao pode receber os valores: 1, 2 e 3

1: construção gulosa

2: construção gulosa + busca local

2: framework ils

O parâmetro seed é opcional e pode ser qualquer valor inteiro
